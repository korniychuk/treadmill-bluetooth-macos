## Сценарии

Отмечай `[x]` по факту. После каждого — `tm doctor` (возраст сэмплов / WARN).

### S1 — Connect + walk

1. Включить дорожку, начать шагать.
2. Ожидание ≤ ~30с: toast found / `tm status` → connected, presence Walking.
3. `tm doctor`: `connected=true`, свежий `last 0x2ACD age`, heartbeat свежий.
4. (опц.) H10: `hr_connected`, live bpm, contact=live.

### S2 — Power-off treadmill while walking (**031 / 035 class**)

1. Во время walk **выключить** дорожку (не pause).
2. ≤ ~20–35с: reconnect / lost toast; `tm doctor` → `connected=false` (не залипает).
3. Лог: `no telemetry received` / session end; **не** «connected» 10+ мин.

### S3 — Remove H10 / freeze contact (**033 / 035**)

1. С H10 на груди: live ♥ в `tm status` / widget.
2. Снять ремень (линк может жить).
3. ≤ ~60с frozen-window: Lost → samples stop; widget ♥ пустой; zone bpm не кормит разгон.
4. (если partial silence) ≤10с silence arm; reconnect tick оживает.

### S4 — `tm zone off` mid-ramp (**032**)

1. Zone on, start walk → phase ramp.
2. `tm zone off` mid-ramp.
3. ≤5с hot-reload: лента **не** продолжает ramp-write; `tm doctor` → config enabled=false, phase/active не engaged (или WARN 032-class).
4. Лог: disengage / no further `control_source=zone` writes.

### S5 — CLI speed vs Zone Hold (**039**)

1. Zone Hold active (Hold), belt moving.
2. `tm speed 2.5` (или своя скорость).
3. ~60с: zone **не** перебивает (лог: `operator override` / no zone set_speed).
4. После окна: zone corrections могут вернуться; все writes с `control_source=…`.

### S6 — Safety floor no-op beeps (**041**)

1. Zone on, belt pinned near `min_speed`, HR выше safety_cap (если безопасно).
2. **Нет** двойного бипа каждые ~5с без смены скорости.
3. Лог: force-reduce только когда target реально меняется.

### S7 — Engage / MORE_DATA sanity (**036**)

1. Старт walk: ramp/default **не** с 0.0 «из ниоткуда».
2. Лог restore/default/zone без подозрительных `to=0` на живой ленте.

### S8 — Doctor only (**038**)

```bash
tm doctor   # daemon up + walking
# power-off belt, wait silence:
tm doctor   # WARN stuck-connected / stale ages if any
```

Без BLE open; exit 0 при читаемой DB.

### S9 — Recompute guard (**044**) — осторожно

```bash
# демон жив → должен отказать:
tm recompute-segments
# ожидается error: stop the daemon first …

# полный recompute — только если готов остановить daemon:
# launchctl bootout gui/$(id -u)/com.korniychuk.treadmill-bluetooth-macos.daemon
# tm recompute-segments
# bash scripts/install-daemon.sh   # вернуть
```

Для smoke достаточно **отказ при живом демоне**.

### S10 — Config atomic write (**037**) — лёгкий

```bash
tm speed-widget off && tm speed-widget on
# config.toml не пустой; daemon hot-reload без mid-session wipe
ls -la ~/.config/treadmill-bluetooth-macos/config.toml
```
